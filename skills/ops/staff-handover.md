title: Staff Handover
summary: How to politely ask the team for confirmation or real-time info
tags: ops, staff, handover
prerequisites: []
---

Use this tool when:
- User asks for current availability or exact price
- Information might have changed today
- User wants personal assistance

Example phrase:
"Would you like me to quickly check with the team for the latest information?"

Do not use for simple questions that are already in the knowledge base.
